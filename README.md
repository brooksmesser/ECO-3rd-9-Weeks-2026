# ECO 3rd Nine Weeks 2026
## Extra Credit Opportunity
To complete this extra credit opportunity, you will need to create a local version of this repo on your github by forking the repository. Complete one, two, or all three functions in order to receive credit.

## Challenge Goals
### Challenge One: The Mining Engine (challenge_one.py)

**Goal**: Recreate the mining mechanic where players can swing their pickaxe to uncover random gemstones. Students must use random probability (a 1-100 dice roll) to determine if a gem is found based on the player's pickaxe tier, and then append the new gem to the player's inventory list.

### Challenge Two: The Shop Menu (challenge_two.py)

**Goal**: Recreate the buying menu for the Curio shop. Students must display a numbered list of available items, allow the player to purchase an item if they have enough money, update the player's pickaxe tier using string slicing, and remove the purchased item from the shop's stock.

### Challenge Three: The Appraisal Engine (challenge_three.py)

**Goal**: Recreate the selling and appraisal mechanic for the Curio shop. Students must build a system that hides the true value of unappraised gems, calculates a random value based on the gem's rarity when appraised, and allows the player to sell the gem for money while removing it from their inventory.

## Grading
For each completed function you will be eligiable for one replaced homework grade of less than your grade for the entire extra credit opportunity. i.e. If you complete one function and get an 80, your lowest grade will be replaced by an 80 if it is not already higher. If you complete three functions for a total grade of a 90, your three lowest grades will be replaced with 90s (if not higher).
